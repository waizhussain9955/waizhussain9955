## 🤖 AI Expertise & Engineering

I build intelligent applications that go beyond standard CRUD operations. My portfolio features a highly advanced **RAG (Retrieval-Augmented Generation) Chatbot** powered by **Cerebras Llama 3.1 8B**.

- **Context-Aware Interactions:** The chatbot dynamically parses database chunks to provide accurate, resume-based answers.
- **Interactive AI UI:** Capable of rendering interactive React components (like Project Cards) directly inside the chat stream.
- **Serverless AI Logic:** Highly optimized AI processing using Next.js API Routes and specialized vector-like string distance matching.
